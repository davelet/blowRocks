# blowRocks 功能愿望清单

> 基于当前项目全部 17 个 C# 脚本的完整分析，梳理已有功能和可扩展方向。
> 优先级定义：P0 = 必做/高价值、P1 = 强烈推荐、P2 = 锦上添花、P3 = 远期/实验性

## 二、核心玩法扩展

### P0 — 必做/高价值

| # | 功能 | 说明 |
|---|------|------|
| 1 | **道具系统（Power-ups）** | 小行星击毁后概率掉落道具：扩散射击、护盾、加速、减速时、额外生命。类似经典 Asteroids 的机制。需要 PowerUp.cs + PowerUpSpawner.cs |
| 2 | **波次系统重构** | 当前只有连续生成；改为清晰波次制：每波固定数量，清完进入下一波，中间有短暂休息，显示 "Wave X" 提示。需要 WaveManager.cs |
| 3 | **分数连击系统** | 连续击杀不中断获得倍率加成（x2, x3...），5秒无击杀则重置。HUD 显示当前倍率 |
| 4 | **玩家死亡动画** | 当前直接 SetActive(false)，缺少视觉反馈。应播放大爆炸粒子 + 碎片 + 屏幕闪烁 |

### P1 — 强烈推荐

| # | 功能 | 说明 |
|---|------|------|
| 5 | **武器升级路线** | 基础单发 → 双发 → 三发散射 → 追踪弹。通过拾取道具临时升级，死亡降一级 |
| 6 | **冲刺/闪避机制** | Shift 键短距离瞬移/冲刺，带冷却条。增加操作深度 |
| 7 | **小行星种类差异化** | 火焰小行星（接触持续伤害）、冰冻小行星（减速玩家）、爆炸小行星（死后范围伤害）、磁性小行星（缓慢追踪玩家） |
| 8 | **UFO 敌人** | 经典 Asteroids 风格：大型/小型 UFO 横穿屏幕并射击玩家，增加威胁层次 |
| 9 | **Boss 战** | 每 5 或 10 波出现 Boss，多血条 + 攻击模式，击败获得大量分数 |
| 10 | **引擎推进物理** | 当前是直接设定速度；改为推力模式：W 加速、有惯性和阻力，更接近经典 Asteroids 手感 |

### P2 — 锦上添花

| # | 功能 | 说明 |
|---|------|------|
| 11 | **地雷/陷阱武器** | 按键放置地雷，小行星碰撞触发范围爆炸 |
| 12 | **护盾能量条** | 独立于生命值的护盾，受击先消耗护盾再扣血，护盾缓慢自动回复 |
| 13 | **子弹类型切换** | 1/2/3 键切换：普通弹（快）、重型弹（慢但伤害高）、散弹（近距离范围） |
| 14 | **有限弹药** | 子弹有弹药上限，拾取补给恢复，迫使玩家精确射击 |
| 15 | **超空间跳跃** | 经典 Asteroids 功能：按 H 键随机传送到屏幕某处，有风险传送到小行星上 |
| 16 | **成就系统** | 首次击杀、连击 x10、一命通关 5 波等。弹出 toast 通知 + 解锁记录 |

### P3 — 远期/实验性

| # | 功能 | 说明 |
|---|------|------|
| 17 | **不同游戏模式** | 无尽模式（当前）、限时模式（3 分钟最高分）、禅模式（无死亡）、一命模式 |
| 18 | **Roguelike 元素** | 每次游戏随机获得不同起始武器/能力，死亡后可选升级 |
| 19 | **剧情关卡** | 预设关卡序列，每关有特定小行星布局和目标 |
| 20 | **本地双人合作** | 分屏或同屏两个玩家，共享生命值 |
